# GEA

This implemntation of Generalized Enrichment Analysis (GEA) can be used to improve the detection of adverse drug events from the biomedical literature.

Requirementes
===================
- R Packages:
	- ROCR
  - data.table


Installation
===================

```
install_github("winnenbr/GEA")
```

Loading
===================

```
library(GEA)
```
